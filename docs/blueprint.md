# **App Name**: ShadowSpeak

## Core Features:

- Unique Code Generation: Generate a unique, single-use code for initiating a new secure chat session.
- Session Joining: Enter a unique code to join an existing secure chat session.
- Ephemeral Messaging: Send and receive encrypted text messages within the secure chat session.
- AI-Powered Expiry: Automatically expire the chat session after a period of inactivity, determined by an AI tool that adjusts the session timeout based on message content and user activity patterns.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to convey security and modernity.
- Background color: Dark charcoal (#222222) for a dark mode theme, reducing eye strain and enhancing privacy.
- Accent color: Deep purple (#800080) to complement the electric blue and add a touch of mystery and sophistication.
- Body and headline font: 'Inter', a sans-serif font, for clear and modern readability.
- Use minimalist icons in electric blue to represent actions and status, ensuring clarity and a consistent aesthetic.
- Clean and intuitive layout, optimized for mobile devices, emphasizing message readability and easy code entry.
- Subtle animations for session initiation, message sending, and session expiry, providing feedback without being distracting.
